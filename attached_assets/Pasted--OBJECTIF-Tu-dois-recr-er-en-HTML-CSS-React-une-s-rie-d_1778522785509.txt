# OBJECTIF

Tu dois recréer en HTML/CSS/React une série de pages inspirées des screenshots fournis, en visant un rendu pixel-perfect desktop.

Il ne s’agit PAS de faire une landing page moderne classique.
Il faut reproduire une direction artistique éditoriale, minimaliste, très espacée, avec une mise en page proche d’un portfolio d’art floral.

Le rendu doit ressembler aux captures fournies :
- fond blanc cassé / gris très clair avec un léger grain
- typographies fines, serrées, parfois condensées
- énormément d’espace vide
- placements asymétriques
- images en blocs rectangulaires
- textes petits, très proches visuellement des captures
- navigation verticale en haut à gauche
- logo manuscrit centré en haut
- approche “magazine / affiche / portfolio”

# RÈGLE IMPORTANTE

Pour le desktop, ne pas utiliser une mise en page fluide classique.
Chaque page doit être construite comme un artboard fixe de 1300px de large par 2048px de haut.

Utiliser principalement :
- position: relative sur chaque page
- position: absolute pour placer précisément les textes et images
- dimensions exactes en px
- line-height très serré
- letter-spacing légèrement négatif
- font-size petit

Ensuite seulement, prévoir une adaptation mobile simplifiée.

# STACK

Utiliser :
- React + Vite
- CSS pur ou Tailwind si nécessaire, mais CSS pur préférable pour la précision
- pas de framework UI
- pas de composants shadcn
- pas d’animations inutiles
- pas de padding générique
- pas de container max-width automatique

# DIMENSIONS GLOBALES

Chaque page :
- width: 1300px
- min-height: 2048px
- background: #f4f4f2 ou proche
- margin: 0 auto
- position: relative
- overflow: hidden

Le body :
- background: #f4f4f2
- color: #151515
- font-smoothing activé
- pas de scroll horizontal

# FOND TEXTURÉ

Ajouter un léger grain sur toute la page via pseudo-element :

body::before ou .page::before :
- position fixed ou absolute
- inset: 0
- pointer-events: none
- opacity: 0.08 à 0.12
- background-image avec radial-gradient ou SVG noise
- mix-blend-mode: multiply

Le rendu doit rester très subtil.

# TYPOGRAPHIES

Le logo manuscrit ne doit PAS être recréé en texte.
Utiliser une image PNG/SVG du logo.

Pour le texte courant :
- utiliser une typo sans-serif très fine et condensée si disponible
- fallback : Helvetica Neue, Arial, sans-serif
- font-weight: 300
- letter-spacing: -0.06em à -0.08em
- line-height: 0.88 à 0.98
- font-size desktop : entre 18px et 22px selon les zones

Pour les titres serif :
- utiliser une serif élégante type Times New Roman / Cormorant Garamond / Editorial New si disponible
- uppercase
- font-size : 18px à 22px
- font-weight: 400
- letter-spacing: -0.02em
- line-height: 1

Important :
Les textes doivent être petits, serrés, presque fragiles.
Ne pas utiliser de gros titres modernes.
